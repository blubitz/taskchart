# taskchart
a task activity timeline
## Usage
### Installation
Download latest version: [v1.0.0-alpha](https://github.com/blubitz/taskchart/releases/download/v1.0.0-alpha/taskchart.app.zip)
### Running locally
1. Clone/download repository
2. run `npm install`
2. run `npm start`
